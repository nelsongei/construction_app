## How to install
- Git clone
- composer install
- cp .env.example .env
- update .env file with your db configurations
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- npm install
- npm run dev
- php artisan serve
- You can use logins from userstable seeder to login on different dashboards
